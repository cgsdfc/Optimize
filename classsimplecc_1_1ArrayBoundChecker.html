Detailed Description

ArrayBoundChecker performs static array bound check on a program.

For any SubscriptExpr in a program, if its index is a simple constant expression then it checks whether the index goes out of bound. Simple constant expression means it is a:

  1. NumExpr.
  2. UnaryOpExpr wrapping a NumExpr, e.g., -1.
  3. NameExpr that is a ConstType. Index other than cases above is excluded from this check.

◆ isConstant()

bool ExprAST::isConstant ( ) const

Return true if this node is mostly a literal value and can be evaluated to int What is mostly a literal value?

  1. CharExpr or NumExpr.
  2. BoolOpExpr, UnaryOpExpr or ParenOpExpr that wraps 1. in their value or operand field.
  3. BinOpExpr is not constant.

Detailed Description

template<typename Derived>
class simplecc::ContextualVisitor< Derived >

ContextualVisitor is a CRTP base for Visitors that knows the LocalSymbolTable of the FuncDef it is visiting.

It sets up the corresponding LocalSymbolTable for each FuncDef to be visited.

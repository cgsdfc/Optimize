Detailed Description

This class represents the ret instruction.

ret is a terminator instruction that terminates the execution of a function. Its type is void. It optionally accepts a Value returned to the caller. If the return Value is void, it is treated as no return Value. Otherwise the type of it must be int. ret int %1 ret void

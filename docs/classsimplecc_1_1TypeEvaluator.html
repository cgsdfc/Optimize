Detailed Description

TypeEvaluator evaluates the result type of an expression.

TypeEvaluator implements the simple rules of the type system. Evaluation of some classes of node requires a LocalSymbolTable so client needs to pass one to its constructor. For example:

int array[10];
typeof(array[2]) is int.
int var;
typeof(var) is int.
const char i = 'c';
typeof(i) is char.
typeof(('a')) is int.
void foo(int i) {}
typeof(foo(0)) is void.
